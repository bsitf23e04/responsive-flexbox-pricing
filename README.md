# Responsive Layouts — Flexbox & Grid Playground

A responsive 3-column pricing layout built using both CSS Flexbox and CSS Grid.

## Features

- Responsive Design
- Mobile Friendly
- Smooth Hover Effects
- Flexbox Layout
- CSS Grid Layout
- Clean Modern UI

## Technologies

- HTML5
- CSS3
- Flexbox
- CSS Grid

## Project Structure

- index.html
- flexbox.html
- grid.html

## Comparison: Flexbox vs Grid

### Flexbox
Pros:
- Easy alignment
- Great for one-dimensional layouts

Cons:
- More effort for complex page structures

### CSS Grid
Pros:
- Ideal for two-dimensional layouts
- Cleaner structure for rows and columns

Cons:
- Slightly harder for beginners
